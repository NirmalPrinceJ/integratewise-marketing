 1. Master Principle

Never treat files as **final outputs**.

Everything should exist as **editable source templates** in:

* **Figma**
* **Google Slides / PowerPoint**
* **Google Docs**

Then you export:

```
Figma / Slides / Docs
        ↓
   PDF / PPT
```

This way you can always **import → edit → update → re-export**.

---

# 2. Importing PowerPoint into Figma (Best Method)

Figma supports **direct PPT import**.

### Steps

1. Open Figma
2. Drag the **.pptx file**
3. Figma converts slides into frames.

Each slide becomes a **fully editable design**.

You can then:

* edit text
* change colors
* replace images
* update layouts

---

# 3. Importing PDF into Figma

PDF import also works.

### Steps

1. Drag PDF into Figma
2. It converts into vector layers

Then you can:

* edit text
* move objects
* replace icons
* redesign slides

Tip:

```
Always export PDF from vector tools
(not flattened images)
```

Otherwise text becomes non-editable.

# 7. Making PDFs Editable Later

Export PDFs like this:

```
Export → PDF → Vector
```

Avoid:

```
Flatten
Rasterize
Outline text
```

This keeps the PDF **editable when re-imported**.
